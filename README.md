# core_framework
A Web light-framework for Lua based Libev

---

## 安装要求

>  支持的操作系统 

    1. linux (most of)

    2. BSD (most of)

    3. Mac OSX

> 依赖库

    1. libev

    2. libssl(openssl/libressl)

    3. jemalloc(optional, 注释掉makefile相关宏即可去除)

    4. lua And version >= 5.3

## 编译方式

> 编译(Only)

    1. Make build

> 重新编译(Clean And Only)

    2. make rebuild

## 运行

    bash#: ./main
